BMalert

A branded replacement for javascript's alert.

To use bmalert, import bmalert's css and js files to your project:

<link rel="stylesheet" type="text/css" href="path/to/bmalert.css">
<script type="text/javascript" src="path/to/bmalert.js"></script>

Then, simply call bmalert's global variable:

bmalert('Hello World!')"

Which can also receive an alert state as a second argument:

bmalert('Error','error')"
bmalert('Warning','warning')"
bmalert('Loading','loading')"
bmalert('Complete','complete')"
bmalert('None','none')"

Or configure the alert to match your needs:


            var bmparams = {
                title: 'Custom Alert',
                alertState: 'complete', // complete|loading|warning|error|none
                cancelButtonText: 'Cancel',
                confirmButtonText: 'OK',
                cancelButtonCallback: function(){},
                confirmButtonCallback: function(){},
                showCancel: false,
                showConfirm: true,
                closeOnConfirm: true,
                clickOutsideToClose: true
            };

            bmalert(bmparams);
